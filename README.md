# The Raven Order

Static website for the Raven Order, published with GitHub Pages.

Live site: https://catzklawz.github.io/raven-order/

## Pages

- `index.html` - Home
- `halls.html` - Founding Charter
- `lore.html` - Constitution
- `gallery.html` - Sigils and Symbols
- `chronicles.html` - Handbook
- `rituals.html` - The Tenets
- `cause.html` - The Cause
- `contact.html` - Contact

## Local development

This site is fully static. Open `index.html` in a browser to preview it locally.

## Deployment

The site is deployed from the `main` branch using GitHub Pages.
